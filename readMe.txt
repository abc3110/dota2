a{blr:expression(this.onFocus=this.blur());}  blr没有特殊意思，expression条件
意思：连接a在获得焦点的同时失去焦点。用的是图片链接，不会显示周边的虚线
1、按下鼠标图片周边不会出现虚线
2、类似于加入script：<a href="#" onfocus="this.blur();">链接标题</a>
